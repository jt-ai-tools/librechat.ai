<Callout type="error" title="錯誤訊息 (UI)" emoji="⚠️">
面向用戶端的錯誤訊息現在會顯示此警告，要求聯繫管理員。

若要查看完整錯誤，請查閱主控台日誌或位於 `./logs` 的額外日誌。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/93c9ad18-c2cb-4862-9e76-546e84671411)

</Callout>

<Callout type="warning" title="日誌位置" emoji="🪵">
  - 完整日誌現在位於 `./logs`（對於本地非 Docker 安裝，仍位於 `./api/logs`）。
</Callout>

<Callout type="warning" title="Google 搜尋外掛程式" emoji="🔍">
  - **[Google 搜尋外掛程式]()**：此外掛程式的環境變數已由 `GOOGLE_API_KEY` 變更為 `GOOGLE_SEARCH_API_KEY`，這是因為與 Google Generative AI 函式庫自動獲取該變數的行為發生衝突。如果您正使用此外掛程式，請相應地更新您的 `.env` 檔案。
</Callout>

<Callout type="info" title="RAG API (檔案對話)" emoji="🗃️">
  - **RAG API 更新**：預設的 Docker Compose 檔案現在包含一個 Python API 與向量資料庫，用以實現 RAG (檢索增強生成)。詳情請參閱 **[RAG API 頁面]()**。
</Callout>
