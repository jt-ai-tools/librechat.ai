## 🌄 亮點介紹

### 🤖 智慧體 (Agents)
- 隆重推出 LibreChat 的「無程式碼智慧體框架 (No-Code Agentic Framework)」
  - 支援在各大供應商上建立自定義 AI 助手，類似於 Assistants API 與 ChatGPT 的 GPTs（[瞭解更多](https://www.librechat.ai/docs/features/agents_zh_TW)）。
  - 支援 Anthropic、OpenAI、AWS Bedrock、Azure OpenAI 以及自定義終端。
- 具備代碼解釋器 (Code Interpreter)、DALL-E-3、Tavily 搜尋等工具強化能力。
- 支援透過 OpenAPI 規範與 MCP 伺服器動態建立工具 ([#4627](https://github.com/danny-avila/LibreChat/pull/4627))。

### 🎯 代碼解釋器 API (Code Interpreter API)
- 透過強大的沙盒環境，安全地執行多種程式語言 ([瞭解更多](https://www.librechat.ai/docs/features/code_interpreter_zh_TW))。
- 具備基於安全工作階段的檔案處理與管理能力 ([#4860](https://github.com/danny-avila/LibreChat/pull/4860))。

### 🔌 MCP 支援 (工具)
- 實作模型上下文協定 (Model Context Protocol) 整合。
  - 它是 「AI 界的 USB-C」，標準化了應用程式如何為 LLM 提供上下文的方式 ([#5015](https://github.com/danny-avila/LibreChat/pull/5015))。
- 更多關於 MCP 與 LibreChat 整合的資訊，請見 [modelcontextprotocol.io](https://modelcontextprotocol.io/clients#librechat)。

### 🚀 新模型支援
- 支援 Gemini 2.0 與 Flash Thinking 模式 ([#4951](https://github.com/danny-avila/LibreChat/pull/4951)，[更多資訊](https://ai.google.dev/gemini-api/docs/thinking-mode))。
- 新增 o1 「完整版」模型，以及 o1-preview/mini 的串流支援 ([#5039](https://github.com/danny-avila/LibreChat/pull/5039))，([#4760](https://github.com/danny-avila/LibreChat/pull/4760))。
- 支援 Claude 3.5 Haiku ([#4629](https://github.com/danny-avila/LibreChat/pull/4629))。
- 支援 AWS Nova 模型 ([#4852](https://github.com/danny-avila/LibreChat/pull/4852))。
