# 更新內容

## 🏞️ 亮點介紹

- 🗨️ 提示詞庫 (Prompts Library)
- 🌿 多重回應串流 (Multi-response Streaming)
- 🔖 對話書籤 (Conversation Bookmarks)
- 🤲 初始的數位無障礙性優化
- 🚀 全面顯著的效能提升

## ✨ 新功能

### 🗨️ 提示詞庫：
* 實作聊天提示詞的建立、分享與使用功能：
  - 包含變數、管理員控制以及針對提示詞的斜線指令 (Slash commands)
  - 由 [@danny-avila](https://github.com/danny-avila) 於 [#3131](https://github.com/danny-avila/LibreChat/pull/3131)、[#3138](https://github.com/danny-avila/LibreChat/pull/3138)、[#3219](https://github.com/danny-avila/LibreChat/pull/3219) 提交

### 🌿 多重回應串流：
* 支援在使用相容的模型/終端時，同時產生 2 則回應
  - 可透過鍵盤指令 (「+」) 或預設設定旁的 (「+」) 按鈕啟動
  - 由 [@danny-avila](https://github.com/danny-avila) 於 [#3191](https://github.com/danny-avila/LibreChat/pull/3191) 提交

### 🔖 對話書籤
* 建立書籤以輕鬆過濾並組織您的對話
* 由 [@ohneda](https://github.com/ohneda)、[@berry-13](https://github.com/berry-13) 與 [@danny-avila](https://github.com/danny-avila) 於 [#3344](https://github.com/danny-avila/LibreChat/pull/3344)、[#3479](https://github.com/danny-avila/LibreChat/pull/3479)、[#3590](https://github.com/danny-avila/LibreChat/pull/3590) 提交

### 其他功能：
* ⚙️ 支援調整訊息串流速率（有助於穩定聊天請求的效能），由 [@danny-avila](https://github.com/danny-avila) 於 [#3244](https://github.com/danny-avila/LibreChat/pull/3244) 提交
* 🔑 為使用者提供的 API 金鑰新增無限期過期選項，由 [@berry-13](https://github.com/berry-13) 於 [#3252](https://github.com/danny-avila/LibreChat/pull/3252) 提交
* ⚡ 原生靜態檔案快取支援，由 [@mawburn](https://github.com/mawburn) 於 [#3455](https://github.com/danny-avila/LibreChat/pull/3455) 提交
