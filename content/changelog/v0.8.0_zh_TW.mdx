## 🏞️ 亮點介紹

自 [v0.7.9](https://www.librechat.ai/changelog/v0.7.9_zh_TW) 以來的主要功能與改進：

### 🔐 細粒度權限系統 (Granular Permissions System)
全面重構權限系統，具備細粒度存取控制、Entra ID 群組發現，以及基於 ACL (存取控制清單) 的資源權限管理，針對智慧體、提示詞、檔案與對話提供權限位元控制。

### 🏪 具備進階分享功能的智慧體市集
建立智慧體市集生態系統，用於發現並分享智慧體，具備分類、推廣系統、包含角色存取控制的進階分享對話框，以及用於使用者/群組/角色搜尋的人員選擇器 UI。

### 🤖 模型更新
- **GPT-5** 支援，具備代幣限制、費率、推理能力與冗長度 (verbosity) 參數控制。
- **Claude Sonnet 4.5** 支援，並更新了 DeepSeek 與 Gemini 最新前沿模型的上下文與計費資訊。

### ⚙️ 其他顯著變動
- 支援子目錄代管 (Subdirectory hosting)
- MCP 相容性與擴充性改進
- 全域應用效能與穩定性提升
- Helm Chart 新增額外配置
- 為 OpenRouter 整合原生網頁搜尋
- MongoDB 連接池配置選項
- [支援以文字上傳 (Upload as Text)](/docs/features/upload_as_text_zh_TW)
- [OpenID Connect](/docs/configuration/authentication/OAuth2-OIDC_zh_TW) 新增額外配置
- 透過 Microsoft Graph API 實作 [SharePoint 整合](/docs/configuration/sharepoint_zh_TW)
