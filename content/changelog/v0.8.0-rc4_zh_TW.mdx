## 更新內容

### ✨ 新功能

* 🌐 支援子目錄 (SubDirectory) 代管，由 [@jpsfs](https://github.com/jpsfs) 於 [#9155](https://github.com/danny-avila/LibreChat/pull/9155) 提交
* 📎 「以文字上傳 (Upload as Text)」支援純文字、音訊、RAG 與代幣限制，由 [@danny-avila](https://github.com/danny-avila) 於 [#8868](https://github.com/danny-avila/LibreChat/pull/8868)、[#9356](https://github.com/danny-avila/LibreChat/pull/9356) 提交
* 🔧 為 AWS ElastiCache TLS 連線新增備用 DNS 查找功能，由 [@MarcAmick](https://github.com/MarcAmick) 於 [#9264](https://github.com/danny-avila/LibreChat/pull/9264) 提交
* 🎨 整合 MCP UI 介面，由 [@samuelpath](https://github.com/samuelpath) 於 [#9299](https://github.com/danny-avila/LibreChat/pull/9299)、[#9418](https://github.com/danny-avila/LibreChat/pull/9418)、[#9581](https://github.com/danny-avila/LibreChat/pull/9581) 提交
* 🔐 新增可配置的最小密碼長度，由 [@danny-avila](https://github.com/danny-avila) 於 [#9315](https://github.com/danny-avila/LibreChat/pull/9315) 提交

* 🔢 支援 JSON Schema 中的整數 (Integer) 與浮點數 (Float) 類型，由 [@danny-avila](https://github.com/danny-avila) 於 [#9469](https://github.com/danny-avila/LibreChat/pull/9469) 提交
* 🗄️ 當餘額功能停用時，允許跳過交易紀錄，由 [@sbruel](https://github.com/sbruel) 於 [#9419](https://github.com/danny-avila/LibreChat/pull/9419) 提交
