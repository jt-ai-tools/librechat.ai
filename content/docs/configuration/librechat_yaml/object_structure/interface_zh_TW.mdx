## 概覽

`interface` 物件讓您可以自定義應用程式中的各種使用者介面元素，包括選單、面板與連結的可見性與行為設定。本章節提供 `interface` 物件結構的詳細解析。

以下是 `interface` 下的欄位：

  - `endpointsMenu` (終端選單)
  - `mcpServers` (MCP 伺服器)
  - `privacyPolicy` (隱私權政策)
  - `termsOfService` (服務條款)
  - `modelSelect` (模型選擇)
  - `parameters` (參數設定)
  - `sidePanel` (側邊欄)
  - `presets` (預設設定)
  - `prompts` (提示詞)
  - `bookmarks` (書籤)
  - `memories` (記憶)
  - `multiConvo` (多重對話)
  - `agents` (智慧體)
  - `temporaryChat` (暫時對話)
  - `temporaryChatRetention` (暫時對話保留時間)
  - `customWelcome` (自定義歡迎訊息)
  - `runCode` (執行程式碼)
  - `webSearch` (網頁搜尋)
  - `fileSearch` (檔案搜尋)
  - `fileCitations` (檔案引用)
  - `peoplePicker` (人員選擇器)
  - `marketplace` (市集)

**備註：**

- `interface` 配置全域套用於應用程式內。
- 大多數設定皆提供預設值，但可根據特定需求或條件進行覆寫。
- 應用程式中的條件邏輯可能會根據其他配置（如模型規範）進一步修改這些設定。

## 範例

```yaml filename="interface"
interface:
  mcpServers:
    placeholder: "MCP 伺服器"
    use: true
    create: true
    share: false
    public: false
    trustCheckbox:
      label: "我信任此伺服器"
      subLabel: "僅啟用您信任的伺服器"
  privacyPolicy:
    externalUrl: "https://example.com/privacy"
    openNewTab: true
  termsOfService:
    externalUrl: "https://example.com/terms"
    openNewTab: true
    modalAcceptance: true
    modalTitle: "服務條款"
    modalContent: |
      # 服務條款
      ## 簡介
      歡迎使用 LibreChat！
  modelSelect: false
  parameters: true
  sidePanel: true
  presets: false
  prompts:
    use: true
    create: true
    share: true
    public: false
  bookmarks: true
  multiConvo: true
  agents:
    use: true
    create: true
    share: true
    public: false
  customWelcome: "嘿 {{user.name}}！歡迎來到 LibreChat"
  runCode: true
  webSearch: true
  fileSearch: true
  fileCitations: true
```

## prompts (提示詞)

**Key：**
<OptionTable
  options={[
    ['prompts', '布林值或物件', '為所有使用者控制提示詞相關功能。可以是簡單啟用/停用的布林值，或針對使用、建立、分享與公開可見性進行細粒度控制的物件。', '設定為 `false` 時，使用者將無法建立、編輯或使用自定義提示詞。'],
  ]}
/>

**預設值：** `true`

**重要：布林值 vs 物件配置**

- **布林值 (`prompts: true`)**：僅更新 `use` (使用) 權限。現有的 `create` (建立)、`share` (分享) 與 `public` (公開) 權限值將從資料庫中 **保留**。這適合用作簡單的功能開關，而不影響透過管理面板進行的其他設定。

- **物件**：僅更新明確指定的子權限。未包含在配置中的權限將從資料庫中保留。

使用物件結構時：

**子鍵 (Sub-keys)：**
<OptionTable
  options={[
    ['use', '布林值', '控制使用者是否可以使用提示詞。', 'true'],
    ['create', '布林值', '控制使用者是否可以建立新提示詞。', 'true'],
    ['share', '布林值', '控制使用者是否可以與特定使用者/群組分享提示詞。', 'false'],
    ['public', '布林值', '控制使用者是否可以公開分享提示詞（對所有使用者可見）。', 'false'],
  ]}
/>

**範例（布林值 - 簡單功能開關）：**
```yaml filename="interface / prompts (布林值)"
interface:
  prompts: true  # 僅更新 USE；create/share/public 維持不變
```

**範例（物件 - 細粒度控制）：**
```yaml filename="interface / prompts (物件)"
interface:
  prompts:
    use: true
    create: false  # 允許使用但停用建立功能
    # share 與 public 未指定 - 保留現有數值
```

**範例（物件 - 完整控制）：**
```yaml filename="interface / prompts (物件)"
interface:
  prompts:
    use: true
    create: true
    share: true
    public: false
```

## temporaryChatRetention (暫時對話保留時間)

`temporaryChatRetention` 配置讓您可以自定義暫時對話在自動刪除前保留的時間長度。

**Key：**
<OptionTable
  options={[
    ['temporaryChatRetention', '數字', '設定暫時對話的保留時間（小時）。', 'temporaryChatRetention: 168'],
  ]}
/>

**驗證規則：**
- **最小值**：1 小時（防止立即刪除）
- **最大值**：8760 小時（最高保留 1 年）
- **預設值**：720 小時（30 天）

**配置方法：**
1. **LibreChat.yaml** (推薦)：`interface.temporaryChatRetention: 168`
2. **環境變數** (已棄用)：`TEMP_CHAT_RETENTION_HOURS=168`

> **備註：** 環境變數 `TEMP_CHAT_RETENTION_HOURS` 已棄用。請改用 `librechat.yaml` 中的 `interface.temporaryChatRetention` 配置選項。配置檔案中的數值優先於環境變數。

**範例：**
```yaml filename="interface / temporaryChatRetention"
interface:
  temporaryChatRetention: 168  # 將暫時對話保留 7 天
```

**常見保留期：**
- **1 小時**：`temporaryChatRetention: 1`（最低保留）
- **24 小時**：`temporaryChatRetention: 24`（1 天）
- **168 小時**：`temporaryChatRetention: 168`（1 週）
- **720 小時**：`temporaryChatRetention: 720`（30 天 - 預設）
- **8760 小時**：`temporaryChatRetention: 8760`（1 年 - 最大值）

## fileCitations (檔案引用)

控制檔案引用功能的全域可用性。停用時，將實際上移除所有使用者的 `FILE_CITATIONS` 權限，不論個別使用者的權限為何，在使用檔案搜尋時都不會顯示任何檔案引用。

**備註：** 
- 此設定作為系統範圍內 `FILE_CITATIONS` 權限的全域開關。
- 設定為 `false` 時，任何使用者都不會看到檔案引用，即使他們透過角色獲得了該權限。
- 檔案引用需要啟用 `fileSearch` 功能。
- 當使用具備檔案搜尋能力的智慧體時，引用的行為（數量與品質）可透過 [Agents 終端配置](/docs/configuration/librechat_yaml/object_structure/agents_zh_TW#檔案引用配置範例)進行配置。

<OptionTable
  options={[

  ]}
/>

**預設值：** `true`

**範例：**
```yaml filename="interface / fileCitations"
interface:
  fileCitations: true
```
