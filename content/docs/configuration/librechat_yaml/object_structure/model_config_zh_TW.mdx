`models` 下的每個項目都是紀錄清單的一部分，可以是布林值或物件：

**當將模型指定為物件時：**

物件形式允許對模型進行詳細配置，包括其 `deploymentName`（部署名稱）及/或 `version`（版本）。此模式用於對模型進行更細粒度的控制，特別是在一個實例或資源群組下處理多個版本或部署時。

**範例**：
```yaml filename="endpoints / azureOpenAI / groups / {group_item} / models / {model_item=Object}"
models:
  gpt-4-vision-preview:
    deploymentName: "gpt-4-vision-preview"
    version: "2024-02-15-preview"
```

**備註：**
- **部署名稱**與**版本**對於確保使用正確的模型至關重要。
    - 請仔細檢查這些數值的準確性，以防止非預期的行為。
