將 Azure OpenAI 服務整合至您的應用程式，讓您可以無縫利用由 Azure OpenAI 託管的多個部署與區域模型。本章節詳細說明如何根據您的需求配置 Azure OpenAI 終端。

**[如需設定 Azure OpenAI 配置的詳細指南，請點擊此處](/docs/configuration/azure_zh_TW)**

## 配置範例

```yaml filename="Azure OpenAI 物件結構範例"
endpoints:
  azureOpenAI:
    titleModel: "gpt-4-turbo"
    groups:
      - group: "my-westus" # 自定義名稱
        apiKey: "${WESTUS_API_KEY}"
        instanceName: "實際實例名稱" # 資源群組或實例的名稱
        version: "2023-12-01-preview"
        # baseURL: https://prod.example.com
        # additionalHeaders:
        #   X-Custom-Header: value
        models:
          gpt-4-vision-preview:
            deploymentName: gpt-4-vision-preview
            version: "2024-02-15-preview"
          gpt-3.5-turbo:
            deploymentName: gpt-35-turbo
          gpt-3.5-turbo-1106:
            deploymentName: gpt-35-turbo-1106
          gpt-4:
            deploymentName: gpt-4
          gpt-4-1106-preview:
            deploymentName: gpt-4-1106-preview
      - group: "my-eastus"
        apiKey: "${EASTUS_API_KEY}"
        instanceName: "實際美國東部實例名稱"
        deploymentName: gpt-4-turbo
        version: "2024-02-15-preview"
        baseURL: "https://gateway.ai.cloudflare.com/v1/cloudflareId/azure/azure-openai/${INSTANCE_NAME}/${DEPLOYMENT_NAME}" # 使用環境變數
        additionalHeaders:
          X-Custom-Header: value
        models:
          gpt-4-turbo: true
```

> **備註：** Azure OpenAI 終端支援所有[共用終端設定](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings_zh_TW)，包括 `streamRate`、`titleModel`、`titleMethod`、`titlePrompt`、`titlePromptTemplate` 與 `titleEndpoint`。

## 群組物件結構 (Group Object Structure)

`groups` 下的每個項目都是紀錄清單的一部分，每個項目包含下列欄位：

### group

**Key：**
<OptionTable
  options={[
    ['group', '字串', '模型群組的識別碼。', ''],
  ]}
/>

**必填：** 是

**範例：**
```yaml filename="endpoints / azureOpenAI / groups / {group_item} / group"
"group": "my-westus"
```

### apiKey

**Key：**
<OptionTable
  options={[
    ['apiKey', '字串', '用於存取 Azure OpenAI 服務的 API 金鑰。', '強烈建議對此欄位使用自定義環境變數引用，例如 `${YOUR_VARIABLE}`。'],
  ]}
/>

**必填：** 是

**範例：**
```yaml filename="endpoints / azureOpenAI / groups / {group_item} / apiKey"
apiKey: "${WESTUS_API_KEY}"
```

### instanceName

**Key：**
<OptionTable
  options={[
    ['instanceName', '字串', 'Azure 實例的名稱。**支援兩種網域格式**：`.openai.azure.com`（舊版）與 `.cognitiveservices.azure.com`（新版）。您可以指定完整網域（例如：`my-instance.cognitiveservices.azure.com`）或僅指定實例名稱（例如：`my-instance`），以向下相容舊版的 `.openai.azure.com` 格式。', '建議對此欄位使用自定義環境變數引用，例如 `${YOUR_VARIABLE}`。'],
