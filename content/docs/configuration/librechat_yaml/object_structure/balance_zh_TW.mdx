## 概覽

`balance` 物件讓管理員可以配置 LibreChat 內使用者的代幣信用餘額管理方式。設定內容包括啟用餘額追蹤、初始化使用者餘額，以及配置自動代幣充值行為。

**`balance` 下的欄位：**

- `enabled` (啟用)
- `startBalance` (初始餘額)
- `autoRefillEnabled` (啟用自動充值)
- `refillIntervalValue` (充值間隔數值)
- `refillIntervalUnit` (充值間隔單位)
- `refillAmount` (充值金額)

**備註：**

- `balance` 配置全域套用於應用程式內。
- 提供預設值，但可根據需求進行自定義。
- 條件邏輯可根據其他配置動態修改這些設定。


## 範例

```yaml filename="balance"
balance:
  enabled: false
  startBalance: 20000
  autoRefillEnabled: false
  refillIntervalValue: 30
  refillIntervalUnit: "days"
  refillAmount: 10000
```
