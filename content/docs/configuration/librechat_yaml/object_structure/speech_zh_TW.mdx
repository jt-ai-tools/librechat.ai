## 概覽

`speech` 物件讓您可以直接在 `librechat.yaml` 配置檔案中配置語音轉文字 (TTS) 與文字轉語音 (STT) 供應商。這啟用了伺服器端語音服務，無需使用者個別配置其 API 金鑰。

**`speech` 下的欄位：**

- `tts` - 文字轉語音供應商配置
- `stt` - 語音轉文字供應商配置
- `speechTab` - 語音功能的預設 UI 設定

**備註：**

- 可同時配置多個供應商。
- 使用者可從可用選項中選擇其偏好的供應商。
- 出於安全考量，配置檔案中的 API 金鑰應使用環境變數引用。

## 範例

```yaml filename="speech"
speech:
  tts:
    openai:
      apiKey: "${TTS_API_KEY}"
      model: "tts-1"
      voices: ["alloy", "echo", "fable", "onyx", "nova", "shimmer"]
    elevenlabs:
      apiKey: "${ELEVENLABS_API_KEY}"
      model: "eleven_multilingual_v2"
      voices: ["voice-id-1", "voice-id-2"]
  stt:
    openai:
      apiKey: "${STT_API_KEY}"
      model: "whisper-1"
  speechTab:
    conversationMode: true
    advancedMode: false
    speechToText: true
    textToSpeech: true
```

```yaml filename="speech / speechTab"
    automaticPlayback: false
    playbackRate: 1.0
    cacheTTS: true
```

## 完整範例

```yaml filename="librechat.yaml"
version: 1.2.9
cache: true

speech:
  tts:
    openai:
      apiKey: "${TTS_API_KEY}"
      model: "tts-1-hd"
      voices: ["alloy", "echo", "fable", "onyx", "nova", "shimmer"]
    elevenlabs:
      apiKey: "${ELEVENLABS_API_KEY}"
      model: "eleven_multilingual_v2"
      voices: ["21m00Tcm4TlvDq8ikWAM", "AZnzlk1XvdvUeBnXmlld"]
      voice_settings:
        stability: 0.5
        similarity_boost: 0.75
  stt:
    openai:
      apiKey: "${STT_API_KEY}"
      model: "whisper-1"
  speechTab:
    conversationMode: false
    advancedMode: false
    speechToText: true
    textToSpeech:
      engineTTS: "openai"
      voice: "nova"
      automaticPlayback: false
```

## 備註

- 配置檔案中的 API 金鑰請務必使用環境變數引用（例如：`${API_KEY}`）。
- 可配置多個 TTS 供應商；使用者可在 UI 中選擇偏好的選項。
- `speechTab` 設定定義了預設值，使用者可在其個人設定中進行覆寫。
- 如需詳細功能說明，請參閱[語音轉文字與文字轉語音設定](/docs/configuration/stt_tts_zh_TW)。
