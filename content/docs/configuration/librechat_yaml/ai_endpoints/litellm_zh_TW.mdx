**備註：**

- 配置請參考 [在 LibreChat 中使用 LiteLLM 代理](/blog/2023-11-30_litellm_zh_TW)。

```yaml filename="librechat.yaml"
    - name: "LiteLLM"
      apiKey: "sk-from-config-file"
      baseURL: "http://localhost:8000/v1"
      # 如果使用 docker-compose.override.yml.example 中的 LiteLLM 範例，請使用 "http://litellm:8000/v1"
      models:
        default: ["gpt-3.5-turbo"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-3.5-turbo"
      summarize: false
      summaryModel: "gpt-3.5-turbo"
      modelDisplayLabel: "LiteLLM"
```
