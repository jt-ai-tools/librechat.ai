> Ollama API 金鑰：必填但會被忽略 - [Ollama OpenAI 相容性說明](https://github.com/ollama/ollama/blob/main/docs/openai.md)

**備註：**

- **已知 (Known)**：已提供圖標。
- 使用 `ollama run` 命令下載模型。參見 [Ollama 模型庫](https://ollama.com/library)。
- 建議在 `titleModel` 中使用 "current_model" 數值，以避免每個對話加載多個模型。
    - 這樣做會動態使用當前的對話模型來產生標題。
- 為了方便起見，此範例包含 Ollama 模型庫中前 5 個熱門模型的清單，最後更新於 2024 年 3 月 1 日。

```yaml filename="librechat.yaml"
  custom:
    - name: "Ollama"
      apiKey: "ollama"
      # 如果在 Docker 容器中執行 LibreChat，請使用 'host.docker.internal' 代替 localhost
      baseURL: "http://localhost:11434/v1/" 
      models:
        default: [
          "llama2",
          "mistral",
          "codellama",
          "dolphin-mixtral",
          "mistral-openorca"
          ]
        # 支援獲取模型列表，但 `name` 欄位必須以 `ollama` 開頭（不區分大小寫），
        # 如此範例所示。
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "Ollama"
```

<Callout type="tip" title="Ollama -> llama3">
    
備註：一旦從 [預設參數](/docs/configuration/librechat_yaml/object_structure/default_params_zh_TW) 中移除了 `stop`，下方提到的問題應不再存在。

然而，如果您遇到 `llama3` 無法停止產生的情況，請在配置中加入此 `addParams` 區塊：

```yaml filename="librechat.yaml"
  custom:
    - name: "Ollama"
      apiKey: "ollama"
      baseURL: "http://host.docker.internal:11434/v1/"
      models:
        default: [
          "llama3"
        ]
        fetch: false # 不支援獲取模型列表
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "Ollama"
      addParams:
          "stop": [
              "<|start_header_id|>",
              "<|end_header_id|>",
              "<|eot_id|>",
              "<|reserved_special_token"
          ]
```

如果您僅在 **Ollama** 中使用 `llama3`，透過 `addParams` 在配置層級設定 `stop` 參數是可以的。

但是，如果您使用多個模型，現在建議透過前端的對話參數與預設 (presets) 加入停止序列。

例如，我們可以省略 `addParams`：

```yaml filename="librechat.yaml"
- name: "Ollama"
    apiKey: "ollama"
    baseURL: "http://host.docker.internal:11434/v1/" 
    models:
    default: [
        "llama3:latest",
        "mistral"
        ]
    fetch: false # 不支援獲取模型列表
    titleConvo: true
    titleModel: "current_model"
    modelDisplayLabel: "Ollama"
```
