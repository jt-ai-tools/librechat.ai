> groq API 金鑰：[wow.groq.com](https://console.groq.com/keys)

**備註：**

- **已知 (Known)**：已提供圖標。

- **溫度 (Temperature)**：如果您將溫度值設定為 0，它將被轉換為 1e-8。如果您遇到任何問題，請嘗試將該值設定為大於 0 且小於或等於 2 的 float32 數值。

- Groq 目前提供免費使用，但有速率限制：每分鐘 10 次請求，每小時 100 次。

```yaml filename="librechat.yaml"
    - name: "groq"
      apiKey: "${GROQ_API_KEY}"
      baseURL: "https://api.groq.com/openai/v1/"
      models:
        default: [
          "llama3-70b-8192",
          "llama3-8b-8192",
          "llama2-70b-4096",
          "mixtral-8x7b-32768",
          "gemma-7b-it",
          ]
        fetch: false
      titleConvo: true
      titleModel: "mixtral-8x7b-32768"
      modelDisplayLabel: "groq"
```
