**備註：**

- **已知 (Known)**：已提供圖標。

- MLX API 對於不被辨識的參數大多要求嚴格。
- 同一時間僅支援一個模型，否則您需要執行另一個具有不同 `baseURL` 的終端。

```yaml filename="librechat.yaml"
    - name: "MLX"
      apiKey: "mlx"
      baseURL: "http://localhost:8080/v1/" 
      models:
        default: [
          "Meta-Llama-3-8B-Instruct-4bit"
          ]
        fetch: false # 不支援獲取模型列表
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "Apple MLX"
      addParams:
            max_tokens: 2000
            "stop": [
              "<|eot_id|>"
            ]
```
