> Fireworks API 金鑰：[fireworks.ai/api-keys](https://fireworks.ai/api-keys)

**備註：**

- **已知 (Known)**：已提供圖標，建議使用動態獲取模型列表 (fetching list of models)。
- 部分模型的 API 可能較為嚴格，不允許包含 `user` 等欄位。在這種情況下，您應使用 [`dropParams`。](/docs/configuration/librechat_yaml/object_structure/custom_endpoint_zh_TW#dropparams)

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false 
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```
