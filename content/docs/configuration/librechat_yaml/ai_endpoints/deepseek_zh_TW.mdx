> Deepseek API 金鑰：[platform.deepseek.com](https://platform.deepseek.com/usage)

**備註：**

- **已知 (Known)**：已提供圖標。
- `deepseek-chat` 與 `deepseek-coder` 相容於 [Agents + 工具](/docs/features/agents_zh_TW)。
- `deepseek-reasoner`（代號 「R1」）已獲得支援，並將串流傳輸其「思考過程 (thought process)」；然而，此特定模型可能不支援某些 OpenAI API 參數。
- 當不使用工具時，「R1」也可能與 Agents 相容。
- 產生標題時建議優先使用 `deepseek-chat`。

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```
