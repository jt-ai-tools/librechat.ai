> **APIpie API 金鑰：** [apipie.ai/dashboard/profile/api-keys](https://apipie.ai/dashboard/profile/api-keys)

**備註：**

- **已知 (Known)**：已提供圖標。建議使用動態獲取模型列表 (fetching list of models)，因為當模型被獲取時，系統會使用 API 代幣費率與定價來計算代幣信用餘額。

- **已知問題：** 
  - 尚不支援獲取模型列表。
  - 對話標題產生的成功率可能有所不同。
  - 目前尚不支援串流 (Stream)（但截至 2024 年 4 月 24 日已在計畫中）。

<Callout type="tip" title="獲取並排序模型" collapsible>
此 Python 腳本可以為您獲取並排序 LLM 模型。輸出結果將儲存在 models.txt 中，其格式旨在讓您更輕鬆地將其加入 YAML 配置。

```py filename="fetch.py"
import json
import requests

def fetch_and_order_models():
    # API 終端
    url = "https://apipie.ai/models"

    # 根據請求範例設定標頭
    headers = {"Accept": "application/json"}

    # 請求參數
    params = {"type": "llm"}

    # 發送請求
    response = requests.get(url, headers=headers, params=params)

    # 解析 JSON 回應
    data = response.json()

    # 提取並排序唯一的模型 ID 列表
    model_ids = sorted(set([model["id"] for model in data]))

    # 將結果寫入文字檔
    with open("models.txt", "w") as file:
        json.dump(model_ids, file, indent=2)

# 執行函式
if __name__ == "__main__":
    fetch_and_order_models()
```
