> NeurochainAI API 金鑰：必填 - [NeurochainAI REST API 文檔](https://app.neurochain.ai/network-integrations/rest-api)

**備註：**

- API 基於 OpenAI API。
- 模型清單持續增加中，此範例可能未包含最新模型。請關注 [NeurochainAI](https://neurochain.ai) 以獲取最新模型。
- 此範例包含了 NeurochainAI 的模型："Mistral-7B-OpenOrca-GPTQ"。

```yaml filename="neurochainai.yaml"
    - name: "NeurochainAI"
      apiKey: "<=已產生的 API 金鑰=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```
