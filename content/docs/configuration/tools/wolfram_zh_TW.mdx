所有對 Wolfram|Alpha API 的呼叫都必須提供一個 AppID。

- 備註：針對一般使用者，Wolfram API 的呼叫限制為每天 100 次，每月 2000 次。

### 建立帳號 
- 造訪：**[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** 建立您的帳號。

### 獲取您的 AppID
- 造訪 **[開發者入口網站 (Developer Portal)](https://developer.wolframalpha.com/portal/myapps/)** 並點擊 「Get an AppID」。
- 選擇 「LLM API」 作為 「API」 類型，並複製產生的金鑰。

#### 將工具加入智慧體 (Agent)

請參閱 [智慧體 (Agents)](/docs/features/agents_zh_TW#工具-tools) 章節，獲取關於如何將工具加入智慧體的更多資訊。
