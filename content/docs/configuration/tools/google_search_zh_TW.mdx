title: Google 搜尋

---

透過外掛程式 (Plugins) 終端，您可以在 GPT 的協助下使用 Google 搜尋來尋找問題的答案！若要開始，您需要獲取 Google 自定義搜尋 API 金鑰以及 Google 自定義搜尋引擎 ID。接著在您的 `.env` 檔案中定義如下：
```env  
GOOGLE_SEARCH_API_KEY="...."  
GOOGLE_CSE_ID="...."  
```  
  
您首先需要建立一個程式化搜尋引擎並獲取搜尋引擎 ID：**[https://developers.google.com/custom-search/docs/tutorial/creatingcse](https://developers.google.com/custom-search/docs/tutorial/creatingcse)**  
  
接著您可以獲取 API 金鑰，點擊此頁面上的 「Get a key」 按鈕：**[https://developers.google.com/custom-search/v1/introduction](https://developers.google.com/custom-search/v1/introduction)**

### 1\. 前往 [程式化搜尋引擎文檔](https://developers.google.com/custom-search/docs/tutorial/creatingcse) 以獲取搜尋引擎 ID。


### 2\. 點擊 「Defining a Programmable Engine in Control Panel」 下方的 「Control Panel」。


點擊登入（若您還沒有 Google 帳號，請建立一個）：

![google_search-1](https://github.com/danny-avila/LibreChat/assets/32828263/51db1a90-c2dc-493c-b32c-821257c27b4e)


### 3\. 註冊新帳號/登入控制面板 (Control Panel)。


登入後，您將被重新導向至控制面板以建立新的搜尋引擎：

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)


### 4\. 建立新的搜尋引擎。


填寫名稱，選擇 「Search the entire web」 (搜尋整個網路) 並點擊 「Create」：

![google_search-3](https://github.com/danny-avila/LibreChat/assets/32828263/c63441fc-bdb2-4086-bb7a-fcbe3d67aef9)


### 5\. 將您的搜尋引擎 ID 複製到 .env 檔案中。

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)


### 6\. 前往 [custom-search 文檔](https://developers.google.com/custom-search/v1/introduction) 以獲取 Google 搜尋 API 金鑰。


### 7\. 點擊 「Get a Key」：

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)


### 8\. 為您的專案命名並同意服務條款。

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)


### 9\. 將您的 Google 搜尋 API 金鑰複製到 .env 檔案中。

![google_search-7](https://github.com/danny-avila/LibreChat/assets/32828263/8170206a-4ba6-40e3-b20e-bdbac21d6695)
