本章節提供詳細的配置指南，協助您在 LibreChat 中設定各種 AI 供應商及其對應的 API 與憑證。

## 終端 (Endpoints) 配置

「終端」一詞指的是您需要設定並與 LibreChat 整合的 AI 供應商、配置或 API。每個終端都有其專屬的配置程序，可能涉及獲取 API 金鑰、憑證或遵循特定的設定說明。

下列指南可協助您配置不同的終端：

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock_zh_TW)**
    - 設定 AWS Bedrock 整合
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic_zh_TW)**
    - 整合 Anthropic AI 模型
- **[OpenAI](/docs/configuration/pre_configured_ai/openai_zh_TW)**
    - 設定 OpenAI API 整合
- **[Google](/docs/configuration/pre_configured_ai/google_zh_TW)**
    - 配置 Google AI 服務
- **[Assistants](/docs/configuration/pre_configured_ai/assistants_zh_TW)**
    - 啟用並配置 OpenAI 的 Assistants

## 自定義終端配置

**[librechat.yaml 配置指南](/docs/configuration/librechat_yaml_zh_TW)** 提供了關於如何在 LibreChat 中配置自定義終端的詳細說明。

除了預設提供的終端外，LibreChat 配置文件還允許您新增並配置自定義終端。這包含了整合如 Ollama、Mistral AI、OpenRouter 以及眾多其他第三方服務。

透過遵循這些配置指南，您可以無縫整合各種 AI 供應商，解鎖其強大能力，並透過多種 AI 模型與服務增強您的 LibreChat 體驗。
