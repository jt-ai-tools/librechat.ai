<Callout type="info" title="備註">
  LibreChat 支援多種檔案儲存後端與一種 CDN 選項。Amazon S3 與 Azure Blob Storage 是物件/檔案儲存解決方案，而 Firebase 是目前唯一具備全球邊緣傳遞功能的真實 CDN 選項。請根據您的部署需求選擇最合適的方案。
</Callout>
