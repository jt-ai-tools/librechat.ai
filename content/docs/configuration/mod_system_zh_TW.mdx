自動審核系統使用評分機制來追蹤使用者的違規行為。當使用者執行如頻繁登入、註冊或傳送訊息等操作時，會累積違規分數。一旦達到設定的閾值，該使用者及其相關 IP 將被暫時封鎖。此系統透過監測並懲罰快速或可疑的活動來確保平台安全。

在生產環境中，您應該部署 Cloudflare 或其他 DDoS 防護方案來真正保護伺服器免受過量請求的攻擊，但這些設定在很大程度上能保護您免受單個或少數惡意行為者針對您的部署實例進行代理攻擊。

**如需更多細節，請參閱此處的使用者指南：[自動審核](/docs/features/mod_system_zh_TW)**

## 設定

下列是用於配置審核系統的所有相關環境變數。請注意，這些變數也可以在 [/.env.example](https://github.com/danny-avila/LibreChat/blob/main/.env.example) 檔案中找到，並應設定在您自己的 `.env` 檔案中。

**備註：** 目前大多數數值是透過 .env 檔案配置的，但它們可能很快會遷移至僅透過 [`librechat.yaml` 配置檔案](/docs/configuration/librechat_yaml/object_structure/config_zh_TW#ratelimits)進行配置。
