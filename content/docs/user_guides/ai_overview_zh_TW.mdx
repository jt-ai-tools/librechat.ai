LibreChat 讓您可以配置並整合各種 AI 供應商、API 及其對應的憑證。這讓您可以根據需求與要求，利用不同的 AI 模型、設定與功能。

## 核心概念

- **終端 (Endpoints)**：終端指特定的 AI 供應商、配置或 API，它決定了聊天請求中可用的模型與設定。範例包括 OpenAI、Google、外掛程式 (Plugins)、Anthropic 等。

- **預設設定 (Presets)**：預設設定是預先儲存的終端、模型與對話設定組合。您可以建立並管理多個預設設定，以因應不同的使用場景。

- **預設終端 (Default Endpoint)**：若您配置了多個終端，您可以指定一個預設終端，以便在開啟新對話時自動使用。

- **預設設定 (Default Preset)**：同理，您可以設定一個預設設定，在開始新對話時自動套用。

## 功能說明

1. **AI 供應商**：透過提供必要的憑證與 API 金鑰，設定各種預先配置好的 AI 供應商。

2. **管理終端**：根據您的需求啟用或停用不同的終端。

3. **建立與管理預設設定**：定義並儲存特定的終端、模型與對話設定組合為預設設定。

4. **設定預設終端與預設設定**：指定預設終端與預設設定，以簡化開啟新對話的流程。這是一段演示影片：**[設定預設預設設定](https://github.com/danny-avila/LibreChat/assets/110412045/bbde830f-18d9-4884-88e5-1bd8f7ac585d)**。

5. **自定義配置**：探索進階配置選項，例如新增如 Ollama、Mistral AI 或 OpenRouter 等自定義終端。
