## 概覽

LibreChat 中的共享連結 (Shareable Links) 讓您可以透過自動產生的連結與 QR Code 與他人分享對話。無論您是想與團隊成員協作、展示 AI 互動過程，還是分享產生的 Artifacts 組件，此功能都讓您只需點擊按鈕即可輕鬆地讓他人檢視您的對話。

## 核心特色

- **輕鬆分享**：只需點擊幾下即可產生共享連結。
- **QR Code 產生**：透過 QR Code 快速分享對話，方便行動裝置存取。
- **對話分支**：接收者可以在共享連結中看到分叉的對話路徑。
- **Artifacts 支援**：共享的對話現在包含產生的 Artifacts 與互動式組件。
- **可自定義外觀**：接收者可以調整語言與主題偏好。
- **連結管理**：集中式儀表板可管理您所有的共享連結。

### 可用選項：

- **更新連結 (Refresh Link)**：為分享的對話產生一個新的 URL。

- **刪除連結 (Delete Link)**：移除共享連結，撤銷所有持有該連結的人之存取權。

## 常見問題

**問：接收者可以回應共享的對話嗎？**
答：不行，共享連結是唯讀的。接收者可以檢視對話，但不能新增訊息或與 AI 互動。

**問：如果我在分享後刪除了對話會發生什麼事？**
答：該共享連結將不再可用，任何存取該連結的使用者都會看到「找不到共享連結」的頁面。

**問：我可以在分享後編輯對話嗎？**
答：共享連結擷取的是建立當下的對話狀態。後續新增的訊息將不會反映在分享視圖中 —— 然而，對現有訊息所做的編輯則會顯示。

**問：共享連結支援離線瀏覽嗎？**
答：不支援，接收者需要有網際網路連線才能檢視共享對話。
