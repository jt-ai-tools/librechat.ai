## 概覽

此功能為您的 LibreChat 伺服器啟用了基於電子郵件的密碼重設功能。您可以將其配置為與各種電子郵件服務配合使用，包括 Mailgun API、Gmail 以及自定義郵件伺服器。

## 核心特色

- 支援多種電子郵件服務：
  - **Mailgun API** —— 推薦用於封鎖 SMTP 連接埠的伺服器
  - **Gmail** 以及其他預定義的 SMTP 服務
  - 具備進階 SMTP 配置的 **自定義郵件伺服器**
- 提供基礎與進階的配置選項
- 為您的 LibreChat 伺服器實現安全的電子郵件密碼重設功能

## 設定選項

- **Mailgun 配置**：使用 Mailgun 的 API 實現可靠的郵件遞送，特別是在 SMTP 連接埠被封鎖時非常有用
- **基礎 SMTP 配置**：使用 Gmail 等預定義服務，僅需最少配置
- **進階 SMTP 配置**：配置通用的 SMTP 服務，或針對任何郵件供應商自定義設定

**如需詳細的設定說明，請參閱配置指南：[電子郵件設定](/docs/configuration/authentication/email_zh_TW)**
