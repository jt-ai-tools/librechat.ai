本文件在建置時使用 [Shiki](https://shiki.matsu.io) 進行語法突顯。

## 基本用法

使用三個反引號並加上語言識別碼來加入程式碼區塊：

````md
```js
console.log('hello, world')
```
````

渲染效果如下：

```js
console.log('hello, world')
```

## 支援的語言

所有 Shiki 支援的語言皆可使用。常見的語言包括：

- `js` / `javascript`
- `ts` / `typescript`
- `jsx` / `tsx`
- `bash` / `sh`
- `json`
- `yaml`
- `python`
- `go`
- `rust`
- `css`
- `html`
- `markdown` / `md`
- `sql`
- `dockerfile`

## 檔案名稱 (Filenames)

您可以為程式碼區塊加入檔案名稱：

````md
```js filename="example.js"
console.log('hello')
```
````

## 行反白 (Line Highlighting)

反白特定行數：

````md
```js {1,3-4}
import React from 'react'

function Hello() {
  return <div>Hello</div>
}
```
````
