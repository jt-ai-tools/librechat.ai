<Carousel autoplay animationDuration="1500" showBullets perView="3">
  <img src="slide1.jpg" alt="Slide 1" />
  <img src="slide2.jpg" alt="Slide 2" />
  <img src="slide3.jpg" alt="Slide 3" />

此範例展示如何使用 `Carousel` 組件及其對應參數：

- `autoplay`：啟用輪播圖自動播放。
- `animationDuration="1500"`：設定動畫持續時間為 1500 毫秒。
- `showControls`：顯示輪播圖的上一個與下一個控制按鈕。
- `showBullets`：顯示輪播圖的點狀導覽。
- `perView="3"`：同時顯示三張投影片。

在 `Carousel` 組件內部，您可以放置任何想顯示為投影片的內容。在此範例中，我們使用 `<img>` 元素並指定圖片路徑。

## 檔案樹 (File Tree)

<Tabs items={['範例','代碼']}>
&nbsp;

<Tabs.Tab>

<FileTree>
  <FileTree.Folder name="src" defaultOpen>
    <FileTree.File name="main.js" active />
    <FileTree.Folder name="components">
      <FileTree.File name="Header.js" />
      <FileTree.File name="Footer.js" />
    </FileTree.Folder>
  </FileTree.Folder>
  <FileTree.File name="package.json" />
</FileTree>

</Tabs.Tab>

<Tabs.Tab>

- `<FileTree.Folder>`：代表一個資料夾。它可以包含嵌套的資料夾或檔案組件。
- `<FileTree.File>`：代表一個檔案。此組件僅需提供 `name` 參數，決定 UI 顯示的檔案名稱。
- `name`：決定 UI 中顯示的檔案或資料夾名稱。
- `defaultOpen`：設定資料夾的初始狀態。若為 true，資料夾在首次渲染時會處於展開狀態。
- `open`：若對資料夾傳遞此參數，它將始終保持開啟，不受使用者互動影響。這在您想維持檔案樹特定區段顯現時很有用。
- `active`：在檔案上使用此參數時，該檔案名稱會被反白。這適合用來向使用者展示當前正在檢視或選取的檔案。

```tsx copy filename="file-tree.tsx"
<FileTree>
  <FileTree.Folder name="src" defaultOpen>
    <FileTree.File name="main.js" active />
    <FileTree.Folder name="components">
      <FileTree.File name="Header.js" />
      <FileTree.File name="Footer.js" />
    </FileTree.Folder>
  </FileTree.Folder>
  <FileTree.File name="package.json" />
</FileTree>
```

## 嵌套分頁 (Nested Tabs)

此範例展示如何使用 `<Tabs>` 與 `<Tabs.Tab>` 組件建立嵌套分頁。外層的 `<Tabs>` 組件有兩個分頁："水果 (Fruit)" 與 "蔬菜 (Vegetable)"。在這些分頁內部，有針對特定水果與蔬菜的嵌套分頁。

`<Tabs>` 組件上的 `items` 參數是一個字串陣列，代表每個分頁的標籤。每個分頁的內容放置在對應的 `<Tabs.Tab>` 組件內。
