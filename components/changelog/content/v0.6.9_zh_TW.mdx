# 更新內容

根據要求更新後的內容如下：

## 🏞️ 亮點介紹

- **LibreChat 生日快樂！** LibreChat 正式滿一週年了！
- **無限捲動 (Infinite Scrolling)：** 向下捲動可載入更多對話，現在已按日期與使用情況排序。
- **Docker**
  - **支援 ARM64v8 建置：** 確保與更多硬體相容，包括 Raspberry Pi 以及其他基於 ARM 的系統。
  - **Docker Hub 映像：** 預建映像現在可以在 [@librechat](https://hub.docker.com/u/librechat) 以及 [@danny-avila](https://github.com/danny-avila?tab=packages&repo_name=LibreChat) 獲取。
- **支援 GPT-4 與 GPT-3.5 Turbo/0125 模型：** 現已支援最新的 GPT 模型。
- **安全性增強：** 持續改進安全措施，以保護使用者數據與隱私。
- **OpenRouter：** 現在會根據 OpenRouter 的所有模型動態使用代幣成本來更新代幣餘額。
- Assistants API 整合將在此版本發布後不久合併至主分支，此版本標誌著重大更新前的「最新穩定版」。
