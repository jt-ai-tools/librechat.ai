# 更新內容

- **更新方式：** 在專案目錄下執行 `npm run update` 進行全新安裝。
  - **先決條件：** 已安裝 git 與 npm。
  - **推薦操作：** 清除瀏覽器的 localStorage 與 cookies。

### ⚠️ 重大變更

根據 [#927](https://github.com/danny-avila/LibreChat/pull/927)，現在必須在您的 .env 檔案中設定 **JWT_REFRESH_SECRET**。

> 同時建議您將 `SESSION_EXPIRY` 設定為較小的值，並設定 `REFRESH_TOKEN_EXPIRY`。
> 預設值：工作階段過期時間：15 分鐘，重新整理權杖過期時間：7 天。
> _請參閱 **[.env.example](https://github.com/danny-avila/LibreChat/blob/1378eb5097b666a4add27923e47be73919957e5b/.env.example#L314)** 以獲取毫秒計算的精確數值。_
