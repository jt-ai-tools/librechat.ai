## [v0.7.7](https://www.librechat.ai/changelog/v0.7.7_zh_TW)

## 🌄 亮點介紹

### 🌟 新模型與推理能力
- 支援 Claude 3.7，具備推理與思考過程串流功能。
- 支援 GPT-4.5。
- 支援 o1-mini 與 o3-mini，具備策略性函式呼叫與 `reasoning_effort` 參數。
- 支援 Anthropic Agents 推理功能，適用於多個供應商，包含 AWS Bedrock。

### 🧠 強化工具與智慧體框架
- 擴展智慧體能力，優化 UI、組件 (Artifact) 管理與推理過程視覺化。
- 新增工具：來自 OpenWeather 的氣象數據、YouTube 影片分析、Flux 圖片產生。
- Actions 支援 OAuth，實現與第三方服務的無縫連線。
- 支援具備現代化介面的 xAI (Grok) 智慧體。

### 🔐 安全性與帳號管理
- 實作兩步驟驗證 (2FA)，支援備份碼與 QR Code。
- 新增 Apple 身份驗證，簡化登入流程。
- 支援 GitHub Enterprise SSO 登入。
- 新增暫時對話 (Temporary chat) 功能，適用於一次性使用場景。

### 🧰 穩健的開發功能
- 支援 Artifact 編輯與下載，優化代碼工作流。
- 代碼解釋器 (Code Interpreter) API 在現有語言基礎上新增對 Rscript 的支援。

### 💻 優化使用者體驗
- 針對訊息編輯與互動提供多項便利性 (Quality-of-Life) 改進。
- 支援透過 URL 查詢參數設定對話，方便分享。
- 全面提升應用程式的無障礙性。
- 透過 i18Next 提供 18 種語言的多國語言支援。
