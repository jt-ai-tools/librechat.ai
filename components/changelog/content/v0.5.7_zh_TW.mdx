# 更新內容

## ✨ 新功能

- 設定與預設設定 (Settings/Presets) UI 重構，並將多個檔案轉換為 TS，由 [@danny-avila](https://github.com/danny-avila) 於 [#740](https://github.com/danny-avila/LibreChat/pull/740) 提交。
  - **此版本包含多項 UI 優化**（變更了超過 200 個檔案），具體變更請見 PR 內容。
    - 整體而言，UI 變得更簡潔，且更貼近官方網站的基礎風格。
    - 針對各類螢幕尺寸的對話框/彈出視窗進行了特別處理，效能較以往有顯著提升。
    - 開發者體驗也獲得極大改善，除了類型安全外，終端設定現在變得非常模組化，讓現有代碼更易於管理，也讓未來整合新終端更加輕鬆。
  - 大部分前端 JS 檔案已轉換為 TS，僅剩最後幾十個檔案，本專案即將全面進入 TypeScript 時代！
- 導覽列切換動畫對齊官方網站，由 [@danny-avila](https://github.com/danny-avila) 於 [#755](https://github.com/danny-avila/LibreChat/pull/755) 提交。
- 捲動行為與按鈕樣式更貼近官方網站風格，由 [@danny-avila](https://github.com/danny-avila) 於 [#761](https://github.com/danny-avila/LibreChat/pull/761) 提交。
- 支援透過電子郵件進行密碼重設，由 [@danorlando](https://github.com/danorlando) 於 [#730](https://github.com/danny-avila/LibreChat/pull/730) 提交。
- 在設定中新增下拉選單以切換語言，由 [@NoahDragon](https://github.com/NoahDragon) 於 [#726](https://github.com/danny-avila/LibreChat/pull/726) 提交。
- 在導覽列中新增搜尋列，由 [@techwithanirudh](https://github.com/techwithanirudh) 於 [#760](https://github.com/danny-avila/LibreChat/pull/760) 提交。
