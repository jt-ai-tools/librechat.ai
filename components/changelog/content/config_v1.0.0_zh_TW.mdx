此初始版本引入了強大的配置架構 (schema)，使用 Zod 進行驗證，旨在以結構化且類型安全的方式管理 API 終端及其相關設定。

參見：[librechat.yaml](/docs/configuration/librechat_yaml_zh_TW)

功能特色：

1. **終端配置架構 (`endpointSchema`)**：
   - **名稱驗證**：確保終端名稱不是預設的 `EModelEndpoint` 數值之一。
   - **API 金鑰**：要求提供字串值作為 API 金鑰識別。
   - **基礎 URL**：要求提供字串值作為終端的基礎 URL。
   - **模型配置**：
     - **預設模型**：要求提供至少包含一個模型的字串陣列。
     - **獲取選項**：選填布林值，用以啟用模型獲取。
   - **額外選填設定**：
     - **對話標題**：選填布林值，用以切換對話標題功能。
     - **標題方法**：選填選項，可在 「completion」 與 「functions」 方法間選擇。
     - **標題模型**：選填字串，用以指定標題產生用的模型。
     - **摘要**：選填布林值，用以啟用摘要功能。
     - **摘要模型**：選填字串，用以指定摘要使用的模型。
     - **強制提示詞**：選填布林值，用以強制納入提示詞。
     - **模型顯示標籤**：選填字串，用以在 UI 顯示中標記模型。

2. **主配置架構 (`configSchema`)**：
   - **版本**：字串，用以指定配置架構版本。
   - **快取**：布林值，用以切換快取機制。
   - **終端**：
     - **自定義終端**：部分套用 `endpointSchema` 的陣列，允許自定義終端配置。
   - 確保嚴格的物件結構，不允許額外屬性。
