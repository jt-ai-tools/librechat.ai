# 更新內容

- 強化 MongoDB-MeiliSearch 同步功能，由 [@danny-avila](https://github.com/danny-avila) 於 [#722](https://github.com/danny-avila/LibreChat/pull/722) 提交。
- 當未指定 `OPENAI_MODELS` 環境變數時，透過 API 獲取模型清單（適用於 ChimeraGPT），由 [@danny-avila](https://github.com/danny-avila) 提交。
- 外掛程式支援反向代理，由 [@danny-avila](https://github.com/danny-avila) 於 [#691](https://github.com/danny-avila/LibreChat/pull/691) 提交。

## 🐛 臭蟲修復

- 解決分詞器 (tokenizer) 管理中的空指標異常問題，由 [@ywkim](https://github.com/ywkim) 於 [#689](https://github.com/danny-avila/LibreChat/pull/689) 提交。
- 修正 Cognito (OpenID) 的 OAuth 問題，由 [@Donavan](https://github.com/Donavan) 於 [#686](https://github.com/danny-avila/LibreChat/pull/686) 提交。
- 修正 `db/indexSync.js` 中的 TypeError 問題及搜尋分頁問題，由 [@danny-avila](https://github.com/danny-avila) 於 [#722](https://github.com/danny-avila/LibreChat/pull/722) 提交。
- 修正一般設定中的清除對話臭蟲，由 [@danny-avila](https://github.com/danny-avila) 於 [#710](https://github.com/danny-avila/LibreChat/pull/710) 提交。
