- 在 [MCP 伺服器](/docs/configuration/librechat_yaml/object_structure/mcp_servers_zh_TW) 中新增了 `chatMenu`。
  - 設定為 `false` 時，會將該 MCP 伺服器從聊天區下拉選單 (MCPSelect) 中排除。
  - 預設值為 `true`（MCP 伺服器會包含在聊天區下拉選單中）。
  - 允許管理員控制哪些 MCP 伺服器出現在聊天介面的下拉選單中。
  - 適用於隱藏主要透過其他介面存取的公用或專業型 MCP 伺服器。
