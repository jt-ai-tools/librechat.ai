## 更新內容

## 🏞️ 亮點介紹

* **👐 重大的無障礙性 (Accessibility) 優化** —— 進行了廣泛的改進以符合 WCAG 標準。
  * 提升螢幕閱讀器支援、鍵盤導覽、焦點管理以及對比度。
* **🏗️ 動態 MCP 伺服器管理** —— 直接從 UI 新增、配置並分享 MCP 伺服器，並具備存取控制功能。
* **📌 釘選您的收藏** —— 將常用的智慧體與模型釘選到側邊欄以便快速存取。
* **🔐 改進後的 MCP 身份驗證** —— 支援舊版 OAuth 以及多項身份驗證修復。
* **⌛ 暫時對話預設值** —— 可在使用者設定中將暫時對話設定為預設模式。
* **🪨 AWS Bedrock 代理支援** —— 為 AWS Bedrock 終端配置代理伺服器設定。
* **📦 最新的 Express.js v5 與 MCP SDK 套件**
