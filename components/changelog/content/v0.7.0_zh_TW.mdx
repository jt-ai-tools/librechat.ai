version: "0.7.0"

- 引入 Anthropic Claude 3 支援及視覺 (Vision) 支援。
- 預設提供預建 Docker 映像，以精簡化部署流程。
- 更新 UI 主題並進行前端優化，主要針對聊天輸入框與訊息編輯功能。
- 為 OpenAI、Azure OpenAI、自定義終端、Anthropic 與 Google 新增「檔案對話 (Chat with Files)」功能（RAG API + 向量資料庫解決方案）。
- 透過 「librechat.yaml」 新增 Azure OpenAI 配置，允許根據模型選取無縫切換區域/部署。
- 為助手實作了餘額追蹤、Azure 支援，修正 API Actions，支援訊息串流、視覺支援及額外工具。
