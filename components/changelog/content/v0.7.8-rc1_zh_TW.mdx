## 更新內容

# 🧪 發布候選版 (Release Candidate) 通知

此版本為發布候選版 (RC)，這代表將不再引入重大新功能。在最終版本發布前，僅會針對剩餘的微小問題或已確認的功能進行修正。此 RC 版提供了一個在本文後續正式發布 v0.7.8 前測試最穩定版本的機會。

# 🏞️ 亮點介紹

- **新 AI 模型**：支援 GPT-4.1, o4-mini, o3, Gemini 2.5 與 Grok 3。
- **[智慧體鏈 (Agent Chain)](/docs/features/agents_zh_TW#智慧體鏈-agent-chain)**：循序式智慧體混合方案，讓多個專門的 AI 能針對複雜任務進行協作。
- **[Mistral OCR 文字提取](/docs/features/ocr_zh_TW)**：從文件中提取文字，並支援 gpt-4o-transcribe 進行音訊轉錄。
- **OpenAI 網頁搜尋模型**：用於存取即時資訊。
- **持久化代碼環境**：代碼解釋器 (Code Interpreter) 檔案現在可根據需求在不同聊天對話間持久保存。
- **雲端儲存**：整合 S3 與 Azure Blob，具備自動重新整理 URL 功能以實現穩健的檔案處理。
- **[OpenAI 圖片工具 (GPT-Image-1)](/docs/features/image_gen_zh_TW)**：用於圖片產生與編輯。
- **UI 更新**：UX、導覽、無障礙性改進與優化。
- **特殊日期/時間變數**：用於提示詞與智慧體的動態變數。
- **[自動餘額充值系統](/docs/configuration/token_usage_zh_TW#餘額配置-balance-configuration)**：以及針對速率限制強化的 Redis 整合。
