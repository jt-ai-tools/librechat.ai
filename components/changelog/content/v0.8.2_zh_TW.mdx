## 更新內容

## 🏞️ 亮點介紹

### 🌊 具備水平擴充能力的 LLM 可恢復串流
為 Redis 模式下的可恢復串流提供跨實例 (cross-replica) 支援，實現生產環境部署的無縫水平擴充。

### 🏗️ 動態 MCP 伺服器管理
直接從 UI 新增、配置並分享 MCP 伺服器，具備完整的存取控制。包含對 API 金鑰驗證、遠端傳輸網域限制以及改進後的 OAuth 處理之支援。

### 📌 釘選您的收藏
將常用的智慧體與模型釘選到側邊欄，以便快速存取。

### 📊 行內 Mermaid 圖表
直接在聊天訊息中渲染 Mermaid 圖表，並優化了 UX 與焦點渲染體驗。

### 👐 重大的無障礙性 (Accessibility) 優化
進行了廣泛的改進以符合 WCAG 標準，提升了螢幕閱讀器支援、鍵盤導覽、焦點管理以及整個應用程式的對比度。

### 🤖 新模型與供應商支援
- **Moonshot Kimi K2** Bedrock 支援
- **Anthropic Beta** 對 Bedrock 的支援
- **Anthropic Vertex AI** 支援
- **Gemini 圖片產生工具** (Nano Banana)
- **Bedrock Guardrails** 護欄支援
- **GPT-5.1 與 GPT-5.2** 代幣定價更新
- **「極高 (Extra High)」推理投入度** OpenAI 參數支援

### ✨ 其他顯著功能
- **Claude 對話匯入器** 支援思考過程匯入
- 程式碼區塊具備 **浮動複製按鈕**
- 新增 **透明聊天標頭** 選項
- 模型規範 (Model Specs) 支援 **自定義群組圖標**
- **OpenRouter 音訊/影片** 檔案上傳支援
- 記憶 LLM 配置支援 **自定義終端**
- 記憶智慧體支援 **Bedrock 供應商**
